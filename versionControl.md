# Version Control Tags

git tag -a stratonea-frontend-starter-v1.0 -m "Stable version 1.0"

## Check Tags

git tag

## Send all Tags

git push origin --tags

## Deleting Branches

<!-- show available branches -->
git branch  

 <!-- move to main -->
git checkout main

<!-- delete branch locally -->
git branch -d dev

<!-- delete branch on github -->
git push origin --delete dev
